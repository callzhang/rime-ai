# data目录文件使用指南

## 📁 目录结构

```
data/
├── 搜狗词库备份_2025_11_27.bin    # 原始备份文件
├── 词库_带词频.txt                  # 完整词库（带词频）
├── 词库_过滤_带词频.txt             # 过滤后的词库（带词频）
├── 词库_converted.txt               # 完整词库（不带词频）
├── 词库_final.txt                   # 过滤后的词库（不带词频）
└── dicts/                           # 常用词词典目录
    ├── english_10000.txt            # 英文常用词（1万）
    ├── english_all.txt              # 英文完整词典（37万+）
    ├── chinese_80000.txt            # 中文词库（7万）
    └── common_words_merged.txt      # 合并的中英文词典（44万+）
```

## 🔄 工作流程

### 步骤1: 导出带词频的词库
```bash
# 从bin文件导出带词频的词库
python3 sogou_export_with_freq.py data/搜狗词库备份_2025_11_27.bin

# 输出: data/词库_带词频.txt
```

### 步骤2: 过滤词库
```bash
# 使用默认词典过滤（自动从dicts目录加载）
python3 filter_dict.py data/词库_带词频.txt --min-freq=10

# 或指定特定词典
python3 filter_dict.py data/词库_带词频.txt --dict=data/dicts/chinese_80000.txt --min-freq=10

# 输出: data/词库_过滤_带词频.txt
```

### 步骤3: 使用过滤后的词库
- `词库_过滤_带词频.txt` - 包含词频信息，可用于分析
- `词库_final.txt` - 纯词条列表，可用于导入其他输入法

## 📊 文件说明

### 原始文件
- **搜狗词库备份_2025_11_27.bin**: 搜狗输入法导出的原始备份文件

### 导出文件
- **词库_带词频.txt**: 
  - 格式: `词条\t词频`
  - 包含所有词条和使用频率
  - 用途: 分析词频、进一步过滤

- **词库_converted.txt**:
  - 格式: 每行一个词条
  - 不包含词频信息
  - 用途: 查看完整词库

### 过滤后的文件
- **词库_过滤_带词频.txt**:
  - 格式: `词条\t词频`
  - 已过滤: 词频<10、单字、常用词、重复字符等
  - 用途: 最终使用的个性化词库（带词频）

- **词库_final.txt**:
  - 格式: 每行一个词条
  - 已过滤: 同上
  - 用途: 导入其他输入法、纯词条列表

### 词典文件（dicts目录）
- **english_10000.txt**: Google 10000常用英文词
- **english_all.txt**: 完整英文词典（370,737个词）
- **chinese_80000.txt**: 中文词库（70,636个词）
- **common_words_merged.txt**: 合并的中英文词典（441,373个词）

## 🎯 常用操作

### 查看词库统计
```bash
# 查看词条数量
wc -l data/词库_带词频.txt
wc -l data/词库_过滤_带词频.txt

# 查看词频最高的词
sort -t$'\t' -k2 -rn data/词库_带词频.txt | head -20
```

### 导出不带词频的版本
```bash
# 从带词频文件提取纯词条
cut -f1 data/词库_过滤_带词频.txt > data/词库_过滤_纯词条.txt
```

### 自定义过滤
```bash
# 不过滤常用词
python3 filter_dict.py data/词库_带词频.txt --no-common --min-freq=10

# 不过滤单字
python3 filter_dict.py data/词库_带词频.txt --no-single --min-freq=10

# 使用自定义词典
python3 filter_dict.py data/词库_带词频.txt --dict=my_dict.txt --min-freq=10
```

## 💡 建议

1. **保留原始文件**: `搜狗词库备份_2025_11_27.bin` 是原始备份，建议保留
2. **使用带词频版本**: `词库_过滤_带词频.txt` 包含词频信息，更有用
3. **定期更新**: 如果词库有更新，重新执行导出和过滤流程
4. **自定义词典**: 可以在 `dicts/` 目录添加更多词典文件
