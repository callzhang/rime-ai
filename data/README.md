# 词库数据目录

此目录用于存放导入导出的词库文件。

## 📁 文件说明

### 原始文件
- **搜狗词库备份_2025_11_27.bin** (2.2M)
  - 原始搜狗词库备份文件
  - 用途: 使用 `sogou_export_with_freq.py` 导出

### 导出文件
- **词库_带词频.txt** (606K, 57,625行)
  - 从bin文件导出的完整词库（带词频）
  - 格式: `词条\t词频`
  - 用途: 使用 `filter_dict.py` 过滤

### 过滤后的文件（最终版本）
- **词库_final_带词频.txt** (47K, 3,972行) ⭐ **推荐使用**
  - 过滤后的词库（带词频）
  - 已过滤: 词频<10、单字、常用词、重复字符等
  - 格式: `词条\t词频`
  - 用途: 最终使用的个性化词库（带词频信息）

- **词库_final.txt** (34K, 3,972行)
  - 过滤后的词库（不带词频）
  - 从 `词库_final_带词频.txt` 自动生成
  - 格式: 每行一个词条
  - 用途: 纯词条列表，可导入其他输入法

### 词典目录 (dicts/)
- **english_10000.txt** - 英文常用词（1万）
- **english_all.txt** - 英文完整词典（37万+）
- **chinese_80000.txt** - 中文词库（7万）
- **common_words_merged.txt** - 合并的中英文词典（44万+）

## 🔄 使用流程

### 1. 导出带词频的词库
```bash
python3 sogou_export_with_freq.py data/搜狗词库备份_2025_11_27.bin
# 输出: data/词库_带词频.txt
```

### 2. 过滤词库
```bash
python3 filter_dict.py data/词库_带词频.txt --min-freq=10
# 输出: 
#   - data/词库_final_带词频.txt (带词频)
#   - data/词库_final.txt (不带词频，自动生成)
```

### 3. 使用过滤后的词库
- **词库_final_带词频.txt** - 包含词频，推荐使用
- **词库_final.txt** - 纯词条列表，可导入其他输入法

## 💡 使用建议

1. **推荐使用**: `词库_final_带词频.txt` - 已过滤常用词，保留个性化词汇（带词频）
2. **导入其他输入法**: `词库_final.txt` - 纯词条列表（从带词频版本自动生成）
3. **查看完整词库**: `词库_带词频.txt` - 包含所有词条和使用频率信息
4. **分析词频**: `词库_带词频.txt` - 包含使用频率信息
