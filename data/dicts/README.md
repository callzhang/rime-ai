# 常用词词典目录

请将下载的常用词词典文件放置在此目录。

## 词典文件格式

- 每行一个词条
- 支持 `#` 开头的注释行
- 支持制表符分隔的格式（只取第一列）
- 文件扩展名建议使用 `.txt`

## 下载方法

1. **手动下载**：
   - 从GitHub或其他资源下载词典文件
   - 将文件保存到此目录（`data/dicts/`）

2. **使用下载脚本**：
   ```bash
   python3 download_dict.py <URL> [文件名]
   ```

3. **词典文件要求**：
   - 文件扩展名：`.txt`
   - 格式：每行一个词条
   - 编码：UTF-8
   - 支持 `#` 开头的注释行

## 使用方法

1. 下载词典文件到此目录
2. 运行过滤脚本，脚本会自动查找并使用词典文件
3. 或使用 `--dict=data/dicts/文件名.txt` 指定特定词典
